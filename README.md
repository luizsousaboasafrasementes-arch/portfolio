# Portfólio de Projetos — Análise de Dados

Portfólio técnico em HTML, CSS e JavaScript puro, criado para apresentar projetos reais de análise de dados, automação operacional, Power BI, SAP e aplicações web.

## Projetos Apresentados

- **Torre de Controle SAP — Vendas e Logística**
- **Relatórios Power BI — KPIs, Colheita e Saldos**
- **Automação CIF — SAP ZSD080, Excel e Power BI**
- **Automação WhatsApp — Monitoramento de Cargas**
- **Simulador de Bonificação — Safra 2026**

## Tecnologias

- HTML5
- CSS3
- JavaScript puro
- Python
- Power BI
- SAP
- Excel
- Node.js
- React
- TypeScript

O projeto está preparado para deploy estático na Vercel. A página principal é `index.html`, com assets locais dentro da pasta `prints/`.

Abra o arquivo `index.html` diretamente no navegador.
