Torre de Controle SAP — Vendas e Logística

Logística, SAP, status operacional e visão gerencial

Problema / objetivo

Automação CIF — SAP ZSD080, Excel e Power BI

Python, Excel, SAP e redução de trabalho manual

Problema / objetivo

Automatizar o fluxo de exportação do SAP ZSD080, atualizar a planilha Monitoramento CIF e alimentar o arquivo All Cultiver usado no Power BI.

Como foi desenvolvido

Script Python com SAP GUI Scripting, automação de janelas Windows, pandas e openpyxl. A lógica filtra frete CIF, IDs válidos, placas preenchidas e janela de data, normalizando colunas e preservando campos manuais.

Resultado / impacto

O log registra execuções com 20 novos pedidos e base de 6.690 registros; a planilha atual analisada contém 6.694 linhas e 24 colunas.

Automação WhatsApp — Monitoramento de Cargas

Painel e robô operacional

Problema / objetivo

Controlar o envio de mensagens para motoristas sem duplicidade no mesmo dia, reduzindo mensagens repetidas durante o acompanhamento de cargas.

Como foi desenvolvido

Aplicação Streamlit que lê CSV, calcula métricas, cria gráficos Plotly e usa Selenium com Chrome para abrir WhatsApp Web. A chave de controle combina motorista, ordem e data.

Resultado / impacto

O painel mostra total monitorado, enviados, pendentes e problemas. A documentação da versão 2.0 registra a regra de no máximo uma mensagem por motorista por dia.

266 cargas monitoradas no print 2.584 envios registrados no print 0 problemas no print
Python Streamlit Pandas Plotly Selenium WhatsApp Web CSV Chrome WebDriver

Simulador de Bonificação — Safra 2026

Aplicação web de cálculo

Problema / objetivo

Simular bonificações por produtor, centro, cultivar e inscrição, usando dados de entrada do ZMM113, classificações e valores de saca por praça.

Como foi desenvolvido

Backend Node.js/Express com leitura de XLSX, autenticação Supabase, sessões, upload de planilhas, regras de classificação, histórico e geração de PDF/DANFE simulado.

Resultado / impacto

A aplicação centraliza cálculo, histórico, rankings, módulos de produtores e unidades, além de relatórios PDF para conferência interna.

Planilha atual: 9.717 linhas Classificações: 4.968 linhas Praças ZMM113: 9.480 linhas
